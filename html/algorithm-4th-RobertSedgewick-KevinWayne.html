Java Fundamentals

  • Primitive data types
    • byte: 8b integers
    • short: 16b integers
    • int: 32b 2's complement
    • long: 64b integers
    • float: 32b real numbers
    • double: 64b real numbers
    • boolean: true or false
    • char: 16b character
  • Statements
    • Declarations
    • Assignment
    • Conditionals
    • Loops
    • Break and Continue
    • Short notations
  • Arrays

    Stores a sequence of values that are all of the same type

  • Static methods

    A sequence of statements

    Stateless function

      Properties of method
    • Arguments are passed by value
    • Method names can be overloaded
    • A method has a single return value but may have multiple return statements
    • A method may have side effects
      Recursion
    • Base case: A conditional statement as the first statement in the program that has a return
    • Subproblems and should not call overlap subproblems
  • Strings

    A sequence of characters

  • Input/Output
    • Command-line arguments
    • Standard input stream
    • Standard output stream
    • Redirection and piping from the commandline
  • Data abstraction
    • Data type is a set of values and a set of operations on those values
    • An Abstract Data Type is a data type whose representation is hidden from the client
    • When using ADT, we focus on the operations specified in the API and pay no attention to the data representation
    • When implementing ADT, we focus on the data, then implement operations on that data
    • A framework for study of algorithms
